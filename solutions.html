🎮 模拟练习 #1：涉政内容识别

改编自 TK-009 真实案例

场景：你是汽车之家论坛审核员。一条评论写道"任大炮也说了同样的话"，回复对象是雷军相关的帖子。机审未命中，DHS 也未提示。
  1. 你的第一步动作是什么？（提示：先确认还是先拦截？）
  2. 你如何判断"任大炮"是否有风险？（提示：你个人知识 + 查询工具）
  3. 如果一审不通过，回扫时需要注意什么？
  4. 你建议在制度上做什么改进来防止类似漏放？
参考答案：1. 先拦截（暂停发布）再确认 2. 搜索"任大炮"→发现是任志强别称，涉政敏感 3. 回扫不能只看字面，要核查人物关联 4. 建立涉政人物别称/代称对照表，纳入培训

🎮 模拟练习 #2：异常数据识别

改编自 TK-003 真实案例

场景：你是基地管理岗。本周数据显示：王小明次品率从 0.18% 升至 0.66%（正常 3 倍），全组次品从 8 条暴增至 37 条（+362%）。日报正常发送，只汇总了数量，未提异常。
  1. 你发现了什么异常？（至少列 3 个信号）
  2. 你作为管理岗应该立刻做什么？（3 个动作）
  3. 日报内容需要怎么改？（具体改法）
  4. 如何防止下个月再次发生？（制度层面）
参考答案：1. 人员次品率飙升×3、全组暴增+362%、日报无异常标注 2. 暂停该人员审核→逐条回查→针对性培训 3. 日报增加人员排名、TOP3次品类型、环比变化 4. 设置自动阈值告警（次品率>0.5%触发）、管理岗 KPI 明确（异常响应≤4h）
